# CodeAlpha
Student Grade Tracker 

Student Grade Tracker is a web-based application developed using HTML, CSS, and JavaScript. The purpose of this project is to manage and analyze student marks efficiently.

The application allows users to:

* Add student names and marks.
* Automatically calculate grades based on marks.
* Display all student records in a structured table.
* Calculate and display average marks, highest marks, lowest marks, and total number of students.
* Delete student records when required.
* Store data using the browser's Local Storage, so the information remains available even after refreshing the page.

Technologies Used:

* HTML for the structure of the webpage.
* CSS for styling and responsive dashboard design.
* JavaScript for functionality, calculations, and data management.

Working Process:

1. The user enters a student's name and marks.
2. The data is stored in an array and saved in Local Storage.
3. The application updates the table dynamically.
4. JavaScript calculates the grade and statistics.
5. The dashboard displays real-time results.

Key Concepts Used:

* Arrays and Objects
* Functions
* DOM Manipulation
* Event Handling
* Local Storage
* Conditional Statements

This project helped me understand how to build an interactive web application, manage data dynamically, and create a user-friendly dashboard interface.
